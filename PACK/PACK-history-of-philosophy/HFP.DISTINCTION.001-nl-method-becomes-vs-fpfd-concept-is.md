# HFP.DISTINCTION.001 — В НЛ метод становится собой ≠ В ФП/ФД понятие уже есть понятие

## Различение

| «Наука логики» | Философия природы / Философия духа |
|---------------|-------------------------------------|
| Метод развивается **вместе с содержанием** | Метод уже **развит** — понятие есть понятие |
| Три формы движения — **результат** НЛ | Три формы — **имплицитны**, поглощены в конкретном понятии |
| Формы снятия **эксплицитны** как формы | Каждое понятие **есть свой** способ снятия |
| Содержание = чистые мыслительные определения | Содержание = реальное бытие Абсолютной идеи |
| Мы наблюдаем, как понятие разрабатывает метод | Мы следуем за имманентным движением понятия |

## Следствие для нормативной реконструкции

Нормативная модель Aufhebung (три момента, три формы) работает на двух уровнях:

| Уровень | Источник | Что задаёт |
|---------|----------|-----------|
| **Абстрактный** (из НЛ) | Три момента, три формы | Необходимые условия любого корректного снятия |
| **Конкретный** (из ФП/ФД) | Имманентное понятие каждой области | Специфические критерии снятия в данном домене |

## Анти-схематистский аргумент

Применять три формы НЛ как классификационную сетку к природе и духу — ошибка Шеллинга («ночь, в которой все коровы серы», Гегель, Предисловие к ФД). Каждое понятие ФП/ФД есть своя имманентная логика движения:
- **Организм (ФП):** снимает конечность через жизнедеятельность — по понятию органической жизни, не по форме «развитие» из НЛ
- **Государство (ФД):** снимает единичные воли в нравственной субстанции — по понятию Sittlichkeit, не по схеме «рефлексия»

**Тест:** «Это понятие само определяет свой способ снятия, или снятие определяется схемой из НЛ?» Само → ФП/ФД-режим. Из схемы → ошибка схематизма.
