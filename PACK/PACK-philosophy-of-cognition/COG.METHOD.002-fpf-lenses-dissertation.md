## 1. Назначение

Применение паттернов FPF к рефлексии собственной работы по диссертации (хронология, трудности, прорывы).

## 2. Линзы

| Код FPF | Для чего |
|---------|----------|
| A.7 Strict Distinction | Категориальные ошибки (объект ≠ предмет, описание ≠ спецификация) |
| A.6.P RPR | Зонтичные понятия без распаковки («снятие» на три момента) |
| A.1.1 BoundedContext | Столкновение контекстов без мостов (формальная vs гегелевская логика) |
| A.4 Temporal Duality | План vs реально выполненное |
| A.10 Evidence Graph | Незакрытые доказательства, трассировка тезисов |
