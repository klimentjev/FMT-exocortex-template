# Черновики (Drafts)

> Личные черновики — тексты из отсортированных заметок, никому не показываем.
> Индекс: `drafts/draft-list.md`. TTL: 7 дней.

## Формат файла

```markdown
---
source_notes: [fleeting-notes YYYY-MM-DD #N, #M]
target: заготовка | pack | рп
created: YYYY-MM-DD
ttl: YYYY-MM-DD
---

# Название

[связный текст из заметок]
```
