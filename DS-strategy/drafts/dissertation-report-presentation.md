# СЛАЙД 4. Февраль: центральный объект — ЭП

**2 февраля** в цепочке научных результатов появился центральный объект — **эпистемическое противоречие**.

**Первый заход:** противоречия в суждениях → конфликт с принципом взрыва в формальной логике (*ex contradictione quodlibet*): из противоречивых посылок формально следует что угодно.

**Ключевое открытие:** ЭП — это **не** одновременное утверждение P и ¬P об одном субъекте суждения. ЭП — это *два разных определения* одного объекта познания. Предмет метода — отношение между **дефинициями**, а не между пропозициями.

---

# СЛАЙД 5. Путь к определению ЭП

**Переключение на уровень определений:**
- Атрибутивно-реляционные родо-видовые дефиниции
- В них: денотат (**Den**), дефиниендум (**Dfd**), дефиниенс (**Dfn**)
- В дефиниенсе: экстенсионал (**Ext**) и интенсионал (**Int**); в нём — род (**IntG**) и видовые признаки (**IntV₁, IntV₂...**)
- Сравнение **Dfn₁** и **Dfn₂** при одном **Dfd**, одном **Den**, одном контексте **С** → отношения: **эквивалентность** → **комплементарность** → **противоречие**

**Определение ЭП (25.02):** логико-семантическое отношение между двумя эксплицитными родо-видовыми сущностными дефинициями одного и того же предмета, при котором их интенсиональные структуры содержат взаимоисключающие сущностные предикаты при сохранении идентичности денотата.

---

# СЛАЙД 6. Примеры трёх отношений

| Отношение | Пример |
|-----------|--------|
| **Эквивалентность** | «Человек — животное, обладающее разумом» / «Человек — разумное животное» — один интенсионал, разная формулировка |
| **Комплементарность** | «Вода — вещество с массой 18 а.е.м.» / «Вода — вещество, кипящее при 100°C» — разные предикаты, один денотат, не противоречат |
| **Противоречие (а)** | «Человек по природе добр» / «Человек по природе зол» — противоположные предикаты в двух дефинициях |
| **Противоречие (б)** | «Сущность задана (природой/творцом)» / «Существование предшествует сущности» — противоречие между *самими определениями* по сути |

ЭП — отношение **между дефинициями**; предмет анализа и снятия — определения.

# СЛАЙД 7. Корневой водораздел

**Центральная ось всей научной работы:**

*Показать возможности и ограничения формальной и содержательной логик для познания.*

- **Формальная логика** — истинностные значения высказываний; безупречна в своей области.
- **Содержательная логика** (диалектическая) — движение понятий, противоречие как источник развития, конкретность как результат снятия; работает там, где формальная умолкает или даёт ложные тревоги (принцип взрыва).

Диссертация **не** противопоставляет логики и не объявляет одну ошибочной. Она **картографирует границу** между ними. **Aufhebung** — процедура на этой границе.

# СЛАЙД 8. Четыре прорыва

1. **Нормативная реконструкция = спецификация, не описание** — «как воспроизвести?» вместо «что имел в виду Гегель?»; инженерные/методические критерии.
2. **ЭП — специфический объект** — два несовместимых определения при одном предмете; формальная логика бессильна; нужен Aufhebung: поднять оба в понятие более высокого уровня конкретности.
3. **Три момента Aufhebung = критерии корректности** — отрицание, сохранение/вычленение, возведение/поднятие; отсутствие одного → не снятие (разрушение, дуализм, пустая абстракция).
4. **Aufhebung в НЛ — структурный принцип** — не «закон диалектики», а живой принцип в каждой категории; три формы движения: переход (Бытие), рефлексия (Сущность), развитие (Понятие) = три типа ЭП и три способа разрешения.
