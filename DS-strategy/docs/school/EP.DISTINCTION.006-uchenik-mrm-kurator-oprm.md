# D.EP.006: Освоение МРМ (ученик) ≠ Освоение преподавания ОМРМ (куратор)

| Траектория ученика (Акд1, опора Лаб2) | Траектория куратора (Акд2, ЖЦ роли) |
|---|---|
| Цель — **освоить МРМ** (метод разумного мышления) | Цель — **освоить преподавание ОМРМ** по регламенту |
| Встреча с образцом, путь к носительству — содержание Лаб2 / Лаб3 | Этапы жизненного цикла роли, передача образца без искажения |
| Не подменяется «учиться преподавать» на этапе ученика | Не подменяется «ещё раз пройти МРМ как ученик» вместо преподавания образца |

**Почему важно:** Смешение контуров даёт ложные критерии готовности (куратор «ещё ученик») и искажение ОМРМ при адаптации «для простоты».

**Тест:** Вопрос к роли — *«что в итоге должен уметь человек?»* Если ответ — *мыслить по методу и достигать ЦС как ученик* → контур ученика. Если ответ — *вести других по образцу метода без искажения регламента* → контур куратора.
