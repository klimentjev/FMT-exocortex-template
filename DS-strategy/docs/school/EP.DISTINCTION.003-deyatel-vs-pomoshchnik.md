# D.EP.003: Деятель ≠ Помощник/Подрядчик

| Деятель | Помощник/Подрядчик |
|---------|-------------------|
| Работает на свой шедевр с наставничеством | Выполняет выданные задания |
| Мотивирован результатом, который останется | Мотивирован выполнением поручения |
| Развивается через работу над шедевром | Накапливает исполнение |
| Критерии успеха — внешний предъявляемый результат | Критерии успеха — выполнено / не выполнено |
| Встречи = разборы (развитие + замер) | Встречи = статусы (отчёт о выполнении) |

**Почему важно:** Фрейм «деятель vs помощник» меняет всё: мотивацию участия, формат встреч, критерии оценки и путь к развитию. В миссионерской организации помощники не удерживаются — деятели остаются.

**Тест:** Есть ли у человека свой шедевр с заранее заданными критериями, над которым он работает при наставничестве? Да → деятель. Ждёт задания и выполняет → помощник.
