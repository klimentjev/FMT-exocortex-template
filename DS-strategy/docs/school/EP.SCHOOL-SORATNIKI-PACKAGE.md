# Ш5: пакет для соратников

## Статус

`frozen-for-send`

## Назначение

Этот файл фиксирует состав отправляемого пакета о школе `ФИЛОСОФ&Я` для соратников. Его задача - не описать школу заново, а удержать границу между отправкой и дальнейшей концептуальной доработкой.

## Адресат

Соратник, который знает о школе в общих чертах, но ещё не живёт внутри внутренней терминологии и не обязан разбираться в истории пересборки документов.

## Состав пакета

1. `EP.SCHOOL-SORATNIKI-LETTER.md` - сопроводительное письмо.
2. `EP.SCHOOL-PROJECT-DESCRIPTION.md` - основной входной документ.
3. `EP.SCHOOL-COMPENDIUM.md` - приложение и полный свод для тех, кому нужна глубина.

## Не входит в пакет первого слоя

- `EP.SCHOOL-CONCEPT-V1.md`
- `EP.SCHOOL-CFT-OBSERVATION-PROTOCOL.md`
- любые новые развороты по `ЛСПМ`, `ЦФТ`, архитектуре контуров и наблюдаемости

Эти материалы могут использоваться как внутренний источник проверки или как последующее приложение, но не как обязательный входной слой для соратника.

## Definition of Done

Пакет считается готовым, когда выполняются все условия:

1. Его можно отправить одним сообщением без устного сопровождения.
2. После чтения основного документа соратник понимает:
   - что такое школа;
   - зачем она существует;
   - как она устроена;
   - какие роли и входы для соратников уже есть;
   - какой следующий шаг возможен после чтения.
3. Основной документ не требует знания внутренней истории сборки.
4. Все новые сильные идеи, не обязательные для отправки, вынесены в `EP.SCHOOL-AFTER-SEND.md`.

## Правило reopen

Повторное открытие пакета допустимо только по двум причинам:

1. адресат не поймёт предложение;
2. в пакете отсутствует обязательный факт.

Причина `я теперь понимаю глубже` не является основанием для `reopen`.
