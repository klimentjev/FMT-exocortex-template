# FM: Прямая продажа членства без пробного опыта

## 1. Паттерн ошибки
Автор делает прямой оффер высокочекового членства тёплой аудитории, пропуская промежуточный шаг — пробный опыт формата.

## 2. Почему это ошибка
Переживаемый продукт (клуб, живой курс, сообщество) имеет барьер «я не знаю, что там происходит изнутри». Этот барьер не закрывается описанием — только опытом. Оффер без опыта получает отказ не из-за цены и не из-за недоверия к автору, а из-за неизвестности формата.

Симптом: аудитория тёплая, оффер явный, конверсия < 1%.

## 3. Антипаттерн → Паттерн

| Антипаттерн | Паттерн |
|-------------|---------|
| Оффер членства → прямая оплата | Пробная сессия → оффер членства |
| «Приходи в клуб за 15к/мес» | «Попробуй одну встречу за 1.5к — если понравится, расскажу про клуб» |
| Описываешь формат → надеешься на конверсию | Даёшь пережить формат → конверсия 30-40% |

## 4. Тест обнаружения
- Аудитория тёплая (знает автора), явный оффер сделан, конверсия < 1% → вероятно, FM
- Спросить: «Может ли человек понять ценность моего продукта без участия?» Нет → нужен пробник
