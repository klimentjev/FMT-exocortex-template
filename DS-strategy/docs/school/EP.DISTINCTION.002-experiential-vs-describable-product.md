# D.EP.002: Переживаемый продукт ≠ Описываемый продукт

| Переживаемый | Описываемый |
|-------------|-------------|
| Ценность раскрывается только через участие | Ценность понятна из описания |
| Оффер не конвертирует без пробного опыта | Оффер конвертирует через убеждение |
| Примеры: клуб, йога, медитация, живой курс | Примеры: книга, PDF, записанный курс |
| Барьер входа: «не знаю, что там» | Барьер входа: цена/доверие |

**Почему важно:** Продажа переживаемого продукта напрямую (без пробного опыта) даёт низкую конверсию даже при тёплой аудитории — потому что описание не закрывает главный барьер.

**Тест:** Может ли человек понять ценность продукта из описания, не участвуя? Да → описываемый. Нет → переживаемый.

**Конверсионное следствие:** Для переживаемого продукта обязателен промежуточный шаг — пробный опыт формата перед основным оффером.
