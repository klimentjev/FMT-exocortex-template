# D.EP.004: Роль как среда развития ≠ Роль как набор задач

| Роль как среда развития | Роль как набор задач |
|------------------------|----------------------|
| Человек входит, чтобы стать определённым типом мыслителя | Человек входит, чтобы выполнять функции |
| Шедевр — внешний результат; тип мышления — личный результат | Результат — выполненные задачи |
| Роль проектируется под развитие конкретной способности | Роль проектируется под закрытие операционной потребности |
| Куратор → понимает метод изнутри; Редактор → видит архитектуру знания | Куратор → проверяет задания; Редактор → структурирует тексты |
| Удерживает людей через рост | Теряет людей при выгорании |

**Почему важно:** Организация, проектирующая роли как среды развития, получает деятелей, которые растут вместе с ней. Организация, проектирующая роли как наборы задач, получает помощников, которые уходят при первом более интересном предложении.

**Тест:** Что изменится в человеке через год работы в этой роли? Если ответ — «тип мышления» → среда развития. Если ответ — «опыт выполнения таких-то задач» → набор задач.
