# Individ

Здесь хранятся отдельные заметки по индивидуальным занятиям с учениками.

## Два типа занятий (не смешивать)

| Тип | Смысл |
|-----|--------|
| **Осведомленность** | Расширение того, что замечается и учитывается: ориентация, контекст, связь с опытом. Акцент на «что видно / что в поле». |
| **Усилие мысли** | Напряжённая мыслительная работа: рассуждение, сцепка понятий, разбор, аргументация. Акцент на «как думаем и к чему приходим». |

Одно занятие обычно ведёт один из типов; если оба — явно разметить этапы в заметке.

Имя файла (без даты в названии):

- `ind-Имя.md` — префикс `ind`, затем имя ученика (латиница или транслит, через дефис при необходимости).

Дату занятия оформлять **разделом урока**: заголовок второго уровня `## YYYY-MM-DD`, внутри — тип, формат, длительность и подразделы (`### Цель`, `### Ход` и т.д.). Следующие занятия с тем же учеником — новый такой же раздел с новой датой в том же файле.
