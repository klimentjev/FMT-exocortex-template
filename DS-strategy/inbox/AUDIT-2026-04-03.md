### Проблемы commit скриптов

**commit.bat:**
```batch
@echo off
cd /d c:\Users\admin\IWE
git commit -m "chore: finalize draft - philosophy as logic of aufhebung for April 2026 conference"
```
✓ Синтаксис правильный, но сообщение статично (не динамическое).

**commit.sh:**
```bash
#!/bin/bash
cd /d c:\Users\admin\IWE    # ← /d работает только в batch, не в bash!
git commit -m "chore: finalize draft - philosophy as logic of aufhebung for April 2026 conference"
```
❌ **ОШИБКА:** `/d` флаг — это batch CMD, не bash. Должно быть:
```bash
#!/bin/bash
cd "c:/Users/admin/IWE" || cd "c:\\Users\\admin\\IWE"
git commit -m "..."
```
