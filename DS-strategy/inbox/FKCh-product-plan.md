# План ФКЧ: от 4 до 20 участников

## Контекст

| Параметр | Сейчас | Цель |
|----------|--------|------|
| Участников клуба | 4 | 20 |
| Доход ФКЧ | ~60-80к/мес | ~300-400к/мес |
| Аудитория | YouTube 1000 / TG 730 / Email 560 | — |
| Старт нового сезона | 1 апреля | — |

**Диагноз:** аудитория есть, явный оффер не конвертирует. Причина — ФКЧ переживаемый продукт. Нужна открытая сессия как пробный опыт перед сезоном.

---

## Воронка

```
БЕСПЛАТНО  →  КВАРТАЛЬНЫЙ ПАК  →  ОТКРЫТАЯ СЕССИЯ  →  КЛУБ
TG/YouTube     2500–3500 руб        1500–2000 руб       15-20к/мес
               Без участия           С вами              С вами
               (пассивный доход)     (конверсия)
```

### Ступень 0 — Бесплатный контент
- Формат: «репортаж из мышления» — не объяснение, а незакрытый вопрос из обсуждения
- Каналы: TG + YouTube Shorts (60 сек: «Платон vs Аристотель по одному вопросу»)
- Мост к паку: «всё это собрано целиком в квартальном паке»
- Критерии нарезок из записей: [FKCh-what-to-clip-free-content.md](FKCh-what-to-clip-free-content.md)

### Ступень 1 — Квартальный пак
- Состав: введение + по каждому автору (фрагмент + карта + синтез + открытый вопрос) + итоговое сравнение + приглашение на сессию
- Цена: 2500–3500 руб
- Начать с Q2 «Что я должен делать?» (Платон + Аристотель + Сенека)
- Синергия: продавать ученикам блока «Аристотель» Эпохального курса (апрель)

### Ступень 2 — Открытая сессия
- Формат: 90 мин онлайн, 10–12 мест, полноценная встреча клуба
- Цена: 1500–2000 руб
- В конце: 5-7 мин про новый сезон — без давления
- Ожидаемая конверсия в клуб: 30–40%

### Ступень 3 — Клуб
- 15–20к/мес, еженедельно
- Антихёрн: ценность сообщества растёт со временем
