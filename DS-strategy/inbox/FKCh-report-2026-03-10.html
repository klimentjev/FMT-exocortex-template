1. Контекст и диагноз

ПараметрСейчасЦель
Участников клуба420

АудиторияYouTube 1000 / TG 730 / Email 560—
Старт нового сезона1 апреля—

Диагноз: Аудитория есть, явный оффер по всем каналам уже делался — конверсия ~0,2% (4 из ~2000). Причина не в охвате и не в доверии. ФКЧ — переживаемый продукт: ценность раскрывается только через участие. Описание не закрывает барьер «не знаю, что там». Нужен промежуточный шаг — пробный опыт формата (открытая сессия) перед основным оффером.

2. Воронка (4 ступени)

БЕСПЛАТНО → КВАРТАЛЬНЫЙ ПАК → ОТКРЫТАЯ СЕССИЯ → КЛУБ TG/YouTube 2500–3500 руб 1500–2000 руб 15–20 тыс/мес Без участия С вами С вами (пассивный доход) (конверсия)

Ступень 0 — Бесплатный контент

  • Формат: «репортаж из мышления» — не объяснение философии, а незакрытый вопрос из обсуждения клуба.
  • Каналы: TG + YouTube Shorts (60 сек: «Платон vs Аристотель по одному вопросу»).
  • Мост к паку: «всё это собрано целиком в квартальном паке».

Ступень 1 — Квартальный пак

  • Состав: введение + по каждому автору (фрагмент + карта + синтез обсуждения + открытый вопрос) + итоговое сравнение трёх позиций + приглашение на открытую сессию.
  • Цена: 2500–3500 руб. Пассивный доход без вашего участия.
  • Первый пак: Q2 «Что я должен делать?» — Платон, Аристотель, Сенека.
  • Синергия: продавать ученикам блока «Аристотель» Эпохального курса (апрель).

Ступень 2 — Открытая сессия

  • Формат: 90 мин онлайн, 10–12 мест, полноценная встреча клуба (чтение вслух + обсуждение + карта понятий).
  • Цена: 1500–2000 руб. В конце: 5–7 мин про новый сезон — без давления.
  • Ожидаемая конверсия в клуб: 30–40%.

Ступень 3 — Клуб

  • 15–20 тыс/мес, еженедельно. Антихёрн: ценность сообщества растёт со временем → уход = потеря накопленного контекста.

Мосты между ступенями

  • Бесплатный контент → Квартальный пак: периодические посты с формулировкой «всё это собрано целиком в квартальном паке» + ссылка. Контент сначала разжигает вопрос, затем предлагает углубиться.
  • Квартальный пак → Открытая сессия: последняя страница пака: «всё это было создано на живых встречах за два часа. Если хочешь пережить формат изнутри — ближайшая открытая встреча [дата]».
  • Открытая сессия → Клуб: последние 5–7 минут встречи — честный рассказ о том, как устроен клуб и новый сезон, без давления. Формулировка: «если хочешь, чтобы такое происходило каждую неделю — напиши мне лично, расскажу детали».

  • 25 мар: старт рекламы нового сезона (уже в плане Strategy.md).
  • Вместе с рекламой: анонс квартального пака Q2.

7. Доменное знание (Pack)

Зафиксировано в PACK-personal для повторного использования.

EP.DISTINCTION.002 — Переживаемый продукт ≠ Описываемый продукт

ПереживаемыйОписываемый
Ценность только через участиеЦенность понятна из описания
Оффер не конвертирует без пробного опытаОффер конвертирует через убеждение
Примеры: клуб, йога, живой курсПримеры: книга, PDF, записанный курс

Тест: Может ли человек понять ценность продукта из описания, не участвуя? Да → описываемый. Нет → переживаемый.

EP.METHOD.003 — Воронка для переживаемых членских продуктов

1. Определение. Четырёхуровневая воронка для продуктов, ценность которых раскрывается через участие (клубы, сообщества, живые курсы). Каждый уровень самостоятелен по ценности и органично ведёт к следующему.

2. Проблема. Прямая продажа высокочекового членства тёплой аудитории даёт низкую конверсию: люди знают автора, но не знают формат изнутри.

ЭлементОписание
ВходыТёплая аудитория (подписчики, читатели); высокочековый переживаемый продукт
ОбработкаПоследовательное снижение барьера входа через 4 уровня
ВыходыКонверсия в платное членство; пассивный доход от контентного продукта

4. Уровни воронки

УровеньФорматЦенность для клиентаЦель уровня
0. Бесплатный контентКанал/блог«Здесь думают иначе»Охват, доверие
1. Пассивный продуктPDF/запись/гайдГлубина без обязательствДоход без участия; создаёт FOMO по живому

3. ЧленствоРегулярный форматПринадлежность + регулярностьОсновной доход

5. Ключевые принципы

  • Каждый уровень продаёт следующий, а не просто даёт контент.
  • Пробник — не демо, а полноценная сессия: человек должен чувствовать, что получил ценность даже без конверсии.
  • Оффер членства — в конце пробника, не в начале: сначала опыт, потом предложение.
  • Антихёрн в членстве: ценность сообщества растёт со временем → уход = потеря накопленного контекста.

6. Мосты между уровнями

EP.FM.001 — Прямая продажа членства без пробного опыта

Failure mode: прямой оффер высокочекового членства тёплой аудитории без пробника. Симптом: конверсия < 1%. Паттерн: пробная сессия → оффер членства.
