# Protocol Open — IWE work culture

После ритуала согласования открыть любую содержательную работу через рамку из `extensions/iwe-work-culture.md`.

## Обязательный минимум перед стартом

1. Назвать текущий `WP`.
2. Зафиксировать роль пользователя в сессии.
3. Зафиксировать метод работы.
4. Сформулировать ожидаемый артефакт и критерии готовности.
5. Если работа больше 3 часов — связать её с результатом месяца.

Если любой из пунктов не ясен, не переходить сразу к исполнению: сначала восстановить рамку.
