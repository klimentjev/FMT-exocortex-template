# Protocol Close — IWE work culture

Перед завершением сессии выполнить закрытие по рамке из `extensions/iwe-work-culture.md`.

## Проверки закрытия

1. Кратко зафиксировать, что сделано по `WP`.
2. Выделить, что узнано по ходу работы.
3. Проверить `Capture-to-Pack`: есть ли знание, которое нужно записать сразу.
4. Зафиксировать, куда именно это знание должно быть положено:
   - правило;
   - `Pack`;
   - `DS`;
   - `memory`.
5. Проверить, нужен ли независимый второй взгляд.
6. Если замечено расхождение с реальностью, сразу предложить конкретный фикс.

Без такого закрытия сессию не считать завершённой.
