# Day Open — IWE work culture

Перед планированием дня применить рамку из `extensions/iwe-work-culture.md`.

## Проверки перед DayPlan

1. Пройти `Открытие дня`: кто пользователь сегодня по ведущим ролям и какие методы будут основными.
2. Спустить фокус по каскаду:
   - стратегия;
   - результаты месяца;
   - рабочие продукты недели;
   - блоки дня.
3. Для каждого значимого блока зафиксировать:
   - `WP`;
   - роль;
   - метод;
   - бюджет;
   - связь с результатом месяца, если блок крупный.
4. Если появляется работа больше 3 часов без связи с результатом месяца, явно пометить её как техдолг, отвлечение или предмет отдельного решения.
