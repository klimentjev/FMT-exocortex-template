# Шпаргалка: как работать в IWE

> Открыть: `Ctrl+P` → `QUICKSTART`

---

## Основные зоны

| Зона | Путь | Для чего |
|------|------|----------|
| Текущий план недели | `DS-strategy/current/WeekPlan W*.md` | РП, задачи, статусы |
| Оперативная память | `FMT-exocortex-template/memory/MEMORY.md` | Блокирующие правила, таблица РП |
| Входящие материалы | `DS-strategy/inbox/` | Загружаемые тексты, черновики, заметки |
| Стратегия | `DS-strategy/docs/Strategy.md` | Куда двигаемся |
| Доменные знания | `PACK-*/` | Source of truth по темам |
| Навигация по всем репо | `FMT-exocortex-template/memory/navigation.md` | Карта файлов и репо |

---

## Как писать в файлы

**Через чат (попросить меня):**
- «добавь в MEMORY.md новый РП»
- «запиши в fleeting-notes.md замечание: ...»
- «обнови WeekPlan»

**Самостоятельно:**
- `Ctrl+P` → начать вводить имя файла → открыть → `Ctrl+S` для сохранения
- Поиск по содержимому: `Ctrl+Shift+F`
- Замена по всей папке: `Ctrl+Shift+H`

**Куда что класть:**

| Что | Куда |
|-----|------|
| Новый РП (рабочий пакет) | `MEMORY.md` → таблица РП **и** `DS-strategy/current/WeekPlan*.md` |
| Контекст задачи (≥2h или ≥2 сессий) | `DS-strategy/inbox/WP-{N}-{slug}.md` |
| Правило для всех репо | `c:\Users\admin\IWE\CLAUDE.md` |
| Правило для одного репо | `<repo>/CLAUDE.md` |
| Доменное знание (паттерн, метод, различение) | Нужный `PACK-*/` |
| Временная заметка / замечание | `DS-strategy/inbox/fleeting-notes.md` |
| Из Telegram | Автоматически → `DS-strategy/inbox/fleeting-notes.md` |

## Минимальный рабочий ритм (ОРЗ)

```
ОТКРЫТИЕ  →  дать задачу в чат
              я проверяю WP Gate (есть ли в плане недели?)
              объявляю: роль / метод / оценка
              вы подтверждаете → начинаем

РАБОТА    →  я выполняю задачу
              на рубежах: «Capture: [что] → [куда]»

ЗАКРЫТИЕ  →  сказать «закрываю» или «всё»
              я: commit + push + обновление памяти + backup
```

**Исключения** (WP Gate не нужен):
- задача ≤15 мин
- вопрос без изменений файлов
- срочный баг-фикс («срочно»)

**Ключевые команды:**

| Что сказать | Что произойдёт |
|-------------|----------------|
| `«заливай»` / `«запуши»` | commit + push без вопросов |
| `«закрываю»` / `«всё»` | полный протокол закрытия |
| `«запиши замечание: X»` | пишу в fleeting-notes.md |
| `«найди документы про X»` | поиск через knowledge-mcp |
